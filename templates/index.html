Automatic Sewer Defect Detection

Reviews sewer videos and uses a computer-vision algorithm to search for defects

About

This is a prototype sewer defect detection application built by the Septic Superstars capstone team. (More about us?) This interface connects to a computer-vision algorithm we trained to detect various defects found in sewage pipes. The model used to find defects is a YOLOv5 classification model. (More details?)
